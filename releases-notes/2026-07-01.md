# Release Notes - 2026-07-01

## PrecisionProDB v2.4.1

- Added VCF manifest input for one-VCF-per-sample population analysis.
- Manifest files use `filepath` as the first column, with optional `sample` and `name_use` columns.
- Manifest VCF conversion is Python-only and can parse VCF files in parallel with `-t/--threads`.
- Updated `PrecisionProDB`, `PrecisionProDB_Sqlite`, and `vcf2mutation` help text for manifest input.
